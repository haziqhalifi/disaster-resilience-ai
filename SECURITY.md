# Security Policy

## Supported Versions

This repository is currently maintained as a hackathon prototype. Security fixes are applied on a best-effort basis.

## Reporting A Vulnerability

Please report suspected vulnerabilities privately to the maintainers.

Do not open a public GitHub issue for sensitive reports.

Include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation, if known

We will acknowledge reports and provide follow-up as quickly as possible.
